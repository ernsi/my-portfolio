Developer Quick Guide

Adding a new pitch: Add an entry to the pitchData object in the script. Then copy a .pitch-card div and set onclick="openPitch('your-id')".

Linking a real Excel file: Set the modelFile field to your hosted .xlsx URL. The download buttons will link to it automatically.

Report content: Edit the reportHTML field — it's raw HTML injected into the detail panel. Write your full analysis there.

Cards without models: Omit the modelHTML field (set to null). The "Excel Model" tab shows a placeholder, and the card won't show the "Model included" badge.

News articles: Copy a .news-card block. The grid auto-flows into 3 columns.
